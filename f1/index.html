Μπα... θα φύγω...

Μπα... θα φύγω... Γιατί άραγε τα ενδιαφέροντά μου, το γράψιμο εν προκειμένω, να επηρεάζει τόσο πολύ @ τη ζωή μου; Ναι, θα φύγω. Δε μπορώ, όπως όλοι οι άλλοι, να γεμίσω τις μέρες μου με ασήμαντα, καθημερινά πράγματα. Δε μπορώ να μην τρώγομαι από την αγωνία, να είμαι ανέμελος @ ευτυχισμένος. Ώρες – ώρες νιώθω την ύπαρξή μου να ξεφεύγει πια από... τα χέρια μου @ από τη σκέψη μου @ να περνάει μέσα στα χαρτιά μου. Αλλά, πάλι, η επιβεβαίωση της ύπαρξής μου, η ύλη, για να επιβεβαιώνεται πρέπει να ανανεώνεται με την αγάπη, αλλιώς... είναι χαμένη ύλη. Όχι, δε θα πάψω να επικοινωνώ άμεσα με τους άλλους, δια ζώσης. Δε θα πάψω να ξέρω πως ν’ αγαπηθώ απ’ τους ανθρώπους. Δε θα πάψω να δίνω στους άλλους απ’ τον εαυτό μου. Δε θα πάψω να είμαι ένας κοινός άνθρωπος, αλλά θα προσπαθήσω κιόλας να γίνω Άνθρωπος... @ ας πονάω για όλα όσα βλέπω... @ ας ματώνω για να βγάλω από τα σωθικά μου αυτά που γράφω στα χαρτιά μου. Τελικά... σας την έσκασα, δεν πρόκειται να φύγω Θα είμ’ εδώ, για να σας φωνάξω δυνατά γεια σας – γεια σας ρέεε...
